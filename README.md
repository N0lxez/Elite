# Elite Studio Landing Page

High-conversion landing page designed for premium web design services.

## Tech Stack

- **Core:** React 19
- **Styling:** TailwindCSS
- **Animations:** Custom Canvas Particle System + CSS Keyframes
- **Icons:** Lucide React

## Deployment

This project is optimized for **Netlify** or **Vercel**.

1. Connect Repository
2. Build Command: `npm run build`
3. Publish directory: `dist`
